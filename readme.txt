== Description ==

Keep track of your Facebook Comments by using this simple plugin.

For more information, check out <a href="http://www.danycode.com/my-hawk/">My Hawk</a>

Plugin Features:

My Hawk get informations from the Facebook graph API e provide to you the facebook comments that belong to your blog.

== Installation ==

1. Upload the My Hawk plugin to your blog and activate it. 
2. Get your facebook comments through: Menu -> My Hawk
